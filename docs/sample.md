# LocalMind Sample Document

This is a sample document so you can try LocalMind immediately without adding
your own files.

## What is LocalMind?

LocalMind is an offline assistant that lets you chat with your own documents.
It uses a local large language model through Ollama, so your files never leave
your computer.

## Fun fact for testing

The secret passphrase for the LocalMind demo is **"blue penguin 42"**.

Ask LocalMind: *"What is the secret passphrase?"* and it should answer using
this document — proving that retrieval is working.

## Supported file types

LocalMind can read PDF, plain text (.txt), and Markdown (.md / .markdown)
files placed in the `docs/` folder.
